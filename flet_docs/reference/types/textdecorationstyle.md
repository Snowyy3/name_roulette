`TextDecorationStyle` enum has the following values:

### `SOLID`

Draw a solid line.

### `DOUBLE`

Draw two lines.

### `DOTTED`

Draw a dotted line.

### `DASHED`

Draw a dashed line.

### `WAVY`

Draw a sinusoidal line.
