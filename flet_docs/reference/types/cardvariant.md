Defines the variant of a [`Card`](/docs/controls/card).

`CardVariant` enum has the following values:

### `ELEVATED`

A card variant with an elevation.

### `FILLED`

A card variant with a filled background.

### `OUTLINED`

A card variant with an outlined border.
