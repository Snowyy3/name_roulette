`TextCapitalization` enum has the following values:

### `NONE`

Do not change entered text.

### `CHARACTERS`

Every entered symbol is capitalized.

### `WORDS`

Capitalize the first letter of every word.

### `SENTENCES`

Capitalize the first letter of every sentence.
