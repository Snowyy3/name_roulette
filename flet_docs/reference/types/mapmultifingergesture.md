`MapMultiFingerGesture` enum has the following values:

### `ALL`

Flag for all multi-finger gestures.

### `NONE`

No multi-finger gestures.

### `PINCH_MOVE`

Flag for pinch move gesture.

### `PINCH_ZOOM`

Flag for pinch zoom gesture.

### `ROTATE`

Flag for rotation gesture.
