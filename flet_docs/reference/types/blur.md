`Blur` class has the following properties:

### `sigma_x`

Horizontal sigma.

### `sigma_y`

Vertical sigma.

### `tile_mode`

The tile mode for the blur.

Value is of type [`BlurTileMode`](/docs/reference/types/blurtilemode).
